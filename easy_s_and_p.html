Easy S&P

I scraped and cleaned data about the S&P 500 from Wikipedia usind Pandas. Then, using SQLAlchemy, I connected to MySQL and wrote the data to table. I also wrote a Python program to fetch, clean, and store data from Yahoo Finance about each individual stock to MySQL. However, due to rate-limiting by Yahoo Finance, this part is incomplete.

Using HTML and JavaScript, I built a front-end that allows users to filter by sector and sort by the ticker, beta, or recommendation score. I connected my database to the frontend using FastAPI. I also wrote methods to create the urls for filtering and sorting. And I defined the table structures through SQLAlchemy.
